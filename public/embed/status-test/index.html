🧪 Statü Gate Testi

packet.status.update — restoran statü değiştirmeden önce eklentiye sorulan allow/deny gate'i (packet.close ayrı sayfada)

Gate kararı

①Panelde paketin statüsünü değiştirmeyi denediğinde eklenti bu kararı döner:

Panelde dene

②Deny → statü değişmez + mesaj. Allow → normal geçer. Pending → statü askıya alınır (eklenti onayı bekleniyor); paket kaydedilir → aşağıdan çöz. (Gate isteği mirror'a düşer.)

Bekleyen gate'i çöz (pending → async)

③Pending ile askıya alınan son paketi burada çöz: Allow → held statü uygulanır; Deny → iptal (statü kalır). (Gerçekte eklenti kurye atayınca otomatik çağırır.)
